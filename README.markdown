# ASCII Heart #
<h2><font color="red">♥</font></h2>
ASCII Heart is a for-fun project I created for _why day that draws an ASCII art heart.  It has one method that takes the size of the heart to draw.  

## Showing the love ##
    gem install 'ascii_heart'
    ascii_heart 20
